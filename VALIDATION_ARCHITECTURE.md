## Key Components

### 1. ValidationService Class

- **Location**: `server/lib/validationService.ts`
- **Responsibility**: Performs all validation checks
- **Methods**:
  - `validate()` - Main entry point
  - 13 private validation methods
  - Helper utilities (format conversion, validation)

### 2. API Endpoint

- **Location**: `server/routes.ts`
- **Endpoint**: `POST /api/validate-config`
- **Responsibility**: HTTP interface to validation service
- **Features**:
  - Schema pre-validation with Zod
  - Error transformation
  - Structured response formatting

### 3. UI Components

- **Location**: `client/src/pages/steps/Step0_ManualConfig.tsx`
- **Components**:
  - Validation button
  - Results display (errors, warnings, suggestions)
  - Stats bar
  - Color-coded sections
